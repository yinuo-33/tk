📖 自动化上架配置指南

Python + Playwright 本地自动化上架，支持 TikTok Shop + Shopee
pip install playwright
playwright install chromium
from playwright.sync_api import sync_playwright
import json, time

products = json.load(open('products.json'))

with sync_playwright() as p:
    browser = p.chromium.launch(headless=False)
    page = browser.new_page()
    page.goto('https://seller.tiktokglobalshop.com')
    input("请手动完成登录后按回车...")

    for product in products:
        page.goto('https://seller.tiktokglobalshop.com/product/add')
        page.fill('#product-name', product['title'])
        page.fill('#price', str(product['price']))
        page.fill('#stock', str(product['stock']))
        page.click('#submit-btn')
        time.sleep(5)
        print(f"已上架: {product['title']}")

    browser.close()
⚠️ 自动化操作请遵守平台条款，勿高频大量操作
