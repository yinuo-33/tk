📖 自动化上架配置指南

此指南帮助你配置本地Python自动化脚本，实现真正的自动上架。

🔧 环境要求

pip install playwright
playwright install chromium

📝 自动上架脚本示例

from playwright.sync_api import sync_playwright
import json, time

products = json.load(open('products.json'))

with sync_playwright() as p:
    browser = p.chromium.launch(headless=False)
    page = browser.new_page()
    
    # 登录TikTok Seller Center
    page.goto('https://seller.tiktokglobalshop.com')
    # 手动完成登录后继续
    input("请手动登录后按回车...")
    
    for product in products:
        # 导航到添加商品页面
        page.goto('https://seller.tiktokglobalshop.com/product/add')
        
        # 填写商品信息
        page.fill('#product-name', product['title'])
        page.fill('#price', str(product['price']))
        page.fill('#stock', str(product['stock']))
        
        # 提交
        page.click('#submit-btn')
        time.sleep(5)  # 等待提交
        
        print(f"✅ 已上架: {product['title']}")
    
    browser.close()
⚠️ 注意：自动化操作可能违反平台使用条款，建议仅用于个人效率提升，不可大量滥用。
